What is Ceta?

Meta-esque application that's exclusively for students

Have you ever looked on Facebook to search for Course Must Knows group chats to learn about a class you’re taking? Have you ever struggled with finding members for a group project because you don’t know a lot of people in your class? Have you had trouble meeting new classmates during the virtual to in-person shift for university learning?

Have no fear, Ceta is here! Ceta is designed to be a course connector platform where students can connect with others taking the same classes as them without having to indirectly scour through Facebook, Instagram, or any other Meta platforms to find friends! Meta with a C serves exclusively to foster interactions amongst students, especially after in-person transitions in the post-pandema era.

  • Connect with people in the same class as you
  • Talk to past students about courses they've taken that you're interested in
  • Gain resources for the course you're taking

Features

What does Ceta offer?

Interest Connect

Introduces user profiles which showcase common interests

Course Directory

Generates a list of peers taking the same classes

Share Resources

Enables users to share past finals or relevant resources for the class they're taking
